udp	 | 	UDP – User Datagram Protocol – Prosty, bezpołączeniowy protokół przesyłania danych bez gwarancji dostarczenia.
dns	 | 	DNS – Domain Name System – Przetwarza nazwy domenowe na adresy IP i odwrotnie.
tcp	 | 	TCP – Transmission Control Protocol – Niezawodny, połączeniowy protokół transmisji danych z kontrolą błędów.
tls	 | 	TLS – Transport Layer Security – Zapewnia szyfrowanie i bezpieczeństwo transmisji w internecie.
http	 | 	HTTP – Hypertext Transfer Protocol – Protokół przesyłania danych w sieci WWW między przeglądarką a serwerem.

snmp	 | 	SNMP – Simple Network Management Protocol – Służy do monitorowania i zarządzania urządzeniami sieciowymi.

icmp	 | 	ICMP – Internet Control Message Protocol – Używany do diagnozowania problemów sieciowych, np. przez ping.

http2	 | 	HTTP/2 – Hypertext Transfer Protocol Version 2 – Ulepszona wersja HTTP oferująca lepszą wydajność i wielowątkowość.
ssh	 | 	SSH – Secure Shell – Zapewnia bezpieczne zdalne logowanie i tunelowanie danych.
smb	 | 	SMB – Server Message Block – Umożliwia udostępnianie plików i drukarek w sieciach Windows.

smtp	 | 	SMTP – Simple Mail Transfer Protocol – Protokół wysyłania wiadomości e-mail między serwerami.
jabber	 | 	Jabber – Extensible Messaging and Presence Protocol (XMPP) – Protokół komunikacji w czasie rzeczywistym (czat, wiadomości).

wudo	 | 	WUDO – Windows Update Delivery Optimization – P2P do aktualizacji Windows przez udostępnianie paczek.
gre	 | 	GRE – Generic Routing Encapsulation – Protokół tunelowania używany do przesyłania pakietów przez VPN.
ftp	 | 	FTP – File Transfer Protocol – Służy do przesyłania plików między klientem a serwerem.
quic	 | 	QUIC – Quick UDP Internet Connections – Nowoczesny, szybki protokół transportowy bazujący na UDP.
netflow	 | 	NetFlow – Network Flow Monitoring Protocol – Technologia monitorowania ruchu sieciowego (głównie Cisco).
nfs	 | 	NFS – Network File System – Umożliwia dostęp do plików w sieciach Unix/Linux.
rdp	 | 	RDP – Remote Desktop Protocol – Protokół zdalnego pulpitu Microsoftu.

valve-a2s	 | 	Valve-A2S – Valve Server Query Protocol – Protokół zapytań serwerów gier Valve (np. Steam).
mdns	 | 	mDNS – Multicast Domain Name System – Umożliwia rozpoznawanie nazw hostów bez DNS w sieci lokalnej.
portmap	 | 	Portmap – Port Mapper (rpcbind) – Mapuje usługi RPC na odpowiednie porty.

redis	 | 	Redis – Remote Dictionary Server – Szybka, oparta na pamięci baza danych typu key-value.
memcached	 | 	Memcached – Memory Caching System – Rozproszona pamięć podręczna obiektów dla aplikacji.
rip	 | 	RIP – Routing Information Protocol – Protokół trasowania oparty na liczbie przeskoków.
bittorrent	 | 	BitTorrent – BitTorrent Protocol – Protokół peer-to-peer do udostępniania plików.
dnp3	 | 	DNP3 – Distributed Network Protocol v3 – Protokół stosowany w automatyce przemysłowej i energetyce.
mysql	 | 	MySQL – My Structured Query Language – Popularny system zarządzania relacyjną bazą danych.
pjl	 | 	PJL – Printer Job Language – Protokół kontroli drukarek (np. kolejka, stan).

oracle	 | 	Oracle – Oracle Database – System relacyjnej bazy danych od firmy Oracle.
